## Adding the ASTRO PDF

Publisher sites block automated download from this environment. To complete the set:

1. Open the [Practical Radiation Oncology PDF link](https://www.practicalradonc.org/action/showPdf?pii=S1879850025001638) in a browser.
2. Save as `astro-2025-who-grade4-glioma.pdf` in this directory.

Both guidelines are open access; no subscription is required.
