# Clinical Guidelines (Radiotherapy Contouring)

Domain knowledge for target-volume definition in high-grade glioma radiotherapy. These notes support VoxLogicA segmentation and contouring workflows (e.g. BraTS-style MRI volumes).

## How to use

- **Current clinical protocol (Europe):** follow [ESTRO–EANO 2023](./glioblastoma-target-volumes.md#estro-eano-2023).
- **Current broad RT guideline (North America):** follow [ASTRO 2025](./glioblastoma-target-volumes.md#astro-2025).
- **Clinical trials (US / NRG):** use the specific NRG / RTOG protocol; do not assume trial margins match either guideline exactly.

## Guideline families

| Region / context | Primary sources |
|------------------|-----------------|
| Europe | EORTC, ESTRO–EANO |
| North America / trials | RTOG, NRG Oncology, ASTRO |

“Latest” and “most widely adopted” are not always the same in glioma contouring. For a contemporary protocol, cite **ASTRO 2025** (broad RT practice) together with **ESTRO–EANO 2023** (contouring-specific detail).
